📌 Overview

SPARK is an ML-enabled mobile application that delivers intelligent parking recommendations and optimized routing within urban environments. The system integrates an ML service with a routing engine to enhance navigation efficiency and improve overall user experience.

⚙️ Key Features

ML-based parking recommendation service

Optimized routing using A* and Bidirectional Search

FastAPI backend integration

Real-time database connectivity with Supabase

Structured data handling and workflow design

🏗️ System Architecture

The application follows a full-stack architecture consisting of:

📱 Mobile Frontend

🔌 Backend API (FastAPI)

🧠 Integrated ML Service

🗄️ Supabase Database

🗺️ Routing Engine (A* and Bidirectional Search)

🛠️ Technologies Used

Mobile Application Development

FastAPI

Supabase

ML Service Integration

A* Algorithm

Bidirectional Search

🎯 Project Objective

The goal of SPARK is to enhance urban parking efficiency by combining ML-driven recommendations with optimized routing strategies, creating a seamless and intelligent navigation experience.
